🔧 Configuração do Firebase

Conecte o sistema ao Firebase para autenticação e banco de dados

1 Criar Projeto no Firebase

  1. Acesse console.firebase.google.com
  2. Clique em "Adicionar projeto"
  3. Nome: clube-do-natural (ou outro nome)
  4. Desative o Google Analytics (opcional) e crie o projeto

2 Ativar Login com Google

  1. No Firebase Console, vá em Authentication → Sign-in method
  2. Clique em "Google" e ative
  3. Preencha o email de suporte do projeto
  4. Salve
  5. IMPORTANTE: Em Authentication → Settings → Authorized domains, adicione:
    clubedonatural.vercel.app e localhost

3 Criar Banco de Dados (Firestore)

  1. No Firebase Console, vá em Firestore Database
  2. Clique em "Criar banco de dados"
  3. Escolha "Modo de produção"
  4. Localização: southamerica-east1 (São Paulo)
