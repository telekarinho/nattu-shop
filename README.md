# Nattu Shop

Plataforma SaaS para lojas de produtos naturais com operacao completa por loja e marketplace centralizado.

## O que este projeto entrega

- Marketplace publico de produtos naturais
- Catalogo individual por loja
- Painel administrativo com pedidos, estoque, caixa, financeiro e relatorios
- Gestao multi-loja com escopo por tenant
- Base para cobranca recorrente de mensalidade dos lojistas

## Posicionamento

O sistema agora funciona como software para terceiros:

- Cada loja tem seu proprio catalogo, estoque, equipe e operacao
- O dono da plataforma monetiza com assinatura mensal
- O cliente final compra no marketplace e pode navegar por loja

## Rodando localmente

```bash
npm install
npm run dev
```

Abra `http://localhost:3000`.
